Projet : Intégration BDDR

Cette SAÉ a été réalisée en binôme dans le cadre de notre formation. Elle avait pour objectif de nous initier à la synchronisation entre un programme Python et une base de données MySQL via EasyPHP-MyAdmin.

Objectifs du projet

  • Comprendre le fonctionnement d’une base de données relationnelle.
  • Réaliser un MCD (Modèle Conceptuel de Données) pour structurer les données.
  • Créer une base de données sous EasyPHP-MyAdmin.
  • Développer une interface utilisateur avec Tkinter en Python.
  • Synchroniser l’application avec la base de données pour l’affichage et la manipulation des données.

Contexte

Le projet portait sur la gestion des données des pompiers du département des Deux-Sèvres. L’application devait permettre d’afficher et de rechercher des informations sur les pompiers, leurs interventions et leurs casernes.

Modèle Conceptuel de Données (MCD)

Le MCD que nous avons conçu nous a permis de visualiser les interactions entre les différentes entités telles que Pompier, Intervention, Véhicule et Caserne.

Application Tkinter

L’interface développée avec Tkinter permettait de :

  • Consulter la base de données des pompiers.
  • Ajouter ou modifier des informations dans la base de données.

Conclusion

Ce projet nous a permis de développer des compétences en modélisation de données, en programmation orientée interface et en interaction entre une application et une base de données. Travailler en binôme nous a également appris à collaborer efficacement sur un projet technique.
