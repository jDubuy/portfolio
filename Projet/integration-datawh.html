Projet : SAE - Intégration de données dans un Data Warehouse

Ce projet a été réalisé en groupe de quatre dans le cadre de notre formation. Il avait pour objectif de nous initier au traitement automatisé de différents types de données, à la création d'une base de données Oracle, ainsi qu’à son exploitation.

Objectifs du projet

  • Mettre en pratique nos compétences en traitement de données.
  • Créer et gérer une base de données Oracle avec Docker Desktop.
  • Automatiser la création du Data Warehouse.
  • Importer les données dans une application de datavisualisation.

Contexte

Nous devions créer un Data Warehouse à partir de données sur des produits de sport vendus dans le monde entier, en respectant le cahier des charges suivant :

  • Suivre le MCD fourni par l'enseignant.
  • Traiter différents types de fichiers (CSV, Excel, Access, TXT).
  • Utiliser un environnement Docker Desktop.
  • Proposer un rendu sous Power BI, composé de dix tableaux de bord.

Difficultés rencontrées

Le plus grand défi de ce projet a été l’appropriation de Docker Desktop, un outil que nous n’avions jamais utilisé auparavant. Sa configuration et sa prise en main ont nécessité un temps d’adaptation non négligeable.

Conclusion

Globalement, ce projet s’est bien déroulé. Mis à part la découverte de Docker, nous n’avons pas rencontré de difficultés majeures. Il nous a permis de renforcer notre maîtrise du traitement de données et de diagnostiquer de nombreuses anomalies classiques que l’on peut rencontrer dans une base de données. Ce fut une expérience enrichissante, tant sur le plan technique que collaboratif.
