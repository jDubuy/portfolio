Projet : Création d’un site web avec interaction de base de données

Ce projet a été réalisé en groupe de trois dans le cadre de notre formation en informatique. Il avait pour objectif de nous initier à la synchronisation entre un site web développé en PHP et une base de données MySQL, via EasyPHP-MyAdmin.

Objectifs du projet

  • Mettre en pratique nos compétences en développement web.
  • Rechercher ou imaginer une base de données pertinente.
  • Créer et gérer cette base de données avec EasyPHP-MyAdmin.
  • Développer le site en équipe en répartissant les tâches via un SharePoint.
  • Implémenter plusieurs interactions entre le site web et la base de données.

Contexte

Nous avions une certaine liberté quant au choix du sujet, tout en respectant un cahier des charges strict :

  • L’utilisateur devait pouvoir ajouter ou modifier certaines informations dans la base de données.
  • Le site devait contenir des graphiques en JavaScript.
  • Un tableau devait se remplir dynamiquement à partir des données de la base.

Le sujet que nous avons choisi était un site répertoriant les statistiques de plusieurs joueurs NBA, permettant de les comparer entre eux. Il était également possible de créer un joueur personnalisé et de simuler sa carrière match par match. Chaque utilisateur possédait son propre compte, stocké en base de données, afin de pouvoir retrouver son joueur personnalisé.

Difficultés rencontrées

Globalement, nous n’avons pas rencontré de problème majeur au niveau de l’organisation interne du groupe ni dans le choix du sujet. En revanche, notre principale difficulté a été notre ambition concernant la partie simulation de carrière : cette fonctionnalité nous a pris énormément de temps, au détriment de la finalisation et de l’optimisation d’autres parties du site.

Conclusion

Je suis globalement satisfait de notre production. Bien que la gestion du temps n’ait pas été optimale, le rendu final et l’investissement personnel de chaque membre du groupe témoignent de notre implication dans ce projet. Je suis fier du travail accompli, tant sur le plan technique que collaboratif.
