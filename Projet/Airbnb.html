Projet : Site Web JavaScript - Airbnb

Ce projet a été réalisé en groupe de quatre dans le cadre de notre formation, dans le cadre de la spécialité VCOD. Il avait pour objectif de mettre en pratique nos compétences en développement JavaScript, ainsi que la mise en ligne d’un site via GitHub Pages.

Objectifs du projet

Nous devions concevoir un site web sur le thème des logements Airbnb en Île-de-France. Plusieurs critères étaient à respecter, notamment l’utilisation exclusive de la bibliothèque Plotly pour la création des graphiques. Le site devait inclure :

  • Une page de tableau de bord contenant plusieurs indicateurs globaux ;
  • Une page avec des cartes choroplèthes ;
  • Une page avec des graphiques de type small multiples.

Réalisation

Pour organiser notre travail, nous avons réparti les pages entre les membres de l’équipe : chacun s’est vu attribuer une page spécifique, et deux d’entre nous ont collaboré sur le tableau de bord. Une fois le développement terminé, nous avons intégré toutes les parties dans un site commun. Pour innover dans notre présentation, nous avons également intégré notre diaporama PowerPoint directement dans le site.

Conclusion

Ce projet nous a permis de consolider nos compétences en JavaScript, en visualisation de données et en gestion de projet en équipe. Malgré un délai très court, nous avons réussi à livrer un site fonctionnel, structuré et interactif. Cette contrainte de temps nous a appris à mieux prioriser nos tâches et à travailler efficacement sous pression.
