Project: AI Digital Twin (LLM & RAG)

This personal project aims to develop a digital twin based on a Large Language Model (LLM). The goal is to create an AI assistant capable of acting and responding like me, relying on my own data via a RAG (Retrieval-Augmented Generation) system.

Project Objectives

  • Develop a custom LLM simulating my responses and personality
  • Set up a RAG architecture to inject my data and context
  • Explore AI agent development frameworks
  • Manage deployment and interaction with the model

Methodology

The project involves the use of state-of-the-art artificial intelligence tools. The system relies on integrating models via API (such as OpenAI or local Ollama). RAG is implemented using vector databases to quickly index and retrieve my personal information, projects, and notes.

All code, architecture, and documentation for this project are available and updated regularly on GitHub.
