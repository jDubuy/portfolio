Project: SAE - Data Integration in a Data Warehouse

This project was completed in a group of four as part of our training. It aimed to introduce us to automated processing of different types of data, creating an Oracle database, and its exploitation.

Project Objectives

  • Practice our data processing skills.
  • Create and manage an Oracle database with Docker Desktop.
  • Automate Data Warehouse creation.
  • Import data into a data visualization application.

Context

We had to create a Data Warehouse from data about sports products sold worldwide, following these requirements:

  • Follow the MCD provided by the instructor.
  • Process different file types (CSV, Excel, Access, TXT).
  • Use a Docker Desktop environment.
  • Provide a Power BI output composed of ten dashboards.

Difficulties Encountered

The greatest challenge of this project was learning Docker Desktop, a tool we had never used before. Its configuration and learning curve required significant adaptation time.

Conclusion

Overall, this project went well. Apart from discovering Docker, we did not encounter major difficulties. It allowed us to strengthen our data processing skills and diagnose many common anomalies found in databases. It was an enriching experience, both technically and collaboratively.
