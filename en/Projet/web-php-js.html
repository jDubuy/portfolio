Project: Creating a Website with Database Interaction

This project was completed in a group of three as part of our computer science training. It aimed to introduce us to synchronization between a website developed in PHP and a MySQL database using EasyPHP-MyAdmin.

Project Objectives

  • Put our web development skills into practice.
  • Research or imagine a relevant database.
  • Create and manage this database with EasyPHP-MyAdmin.
  • Develop the website as a team by sharing tasks via SharePoint.
  • Implement multiple interactions between the website and the database.

Context

We had some freedom in choosing the topic, while respecting strict requirements:

  • Users should be able to add or modify certain information in the database.
  • The website must contain charts in JavaScript.
  • A table must dynamically populate from database data.

The topic we chose was a website listing statistics for several NBA players, allowing them to be compared. It was also possible to create a custom player and simulate their career match by match. Each user had their own account, stored in the database, so they could retrieve their custom player.

Difficulties Encountered

Overall, we did not encounter major problems in our internal group organization or in the topic choice. However, our main difficulty was our ambition regarding the career simulation feature: this functionality took us a lot of time, at the expense of finalizing and optimizing other parts of the website.

Conclusion

I am generally satisfied with our production. Although time management was not optimal, the final result and each team member's personal investment demonstrate our commitment to this project. I am proud of the work accomplished, both technically and collaboratively.
