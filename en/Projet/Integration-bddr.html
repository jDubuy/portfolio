Project: Database Integration

This SAE was completed in a duo as part of our training. It aimed to introduce us to synchronization between a Python program and a MySQL database via EasyPHP-MyAdmin.

Project Objectives

  • Understand how a relational database works.
  • Create a CDM (Conceptual Data Model) to structure data.
  • Create a database under EasyPHP-MyAdmin.
  • Develop a user interface with Tkinter in Python.
  • Synchronize the application with the database for data display and manipulation.

Context

The project focused on managing fire department data from the Deux-SÃ¨vres department. The application had to display and search for information about firefighters, their interventions, and their fire stations.

Conceptual Data Model (CDM)

The CDM we designed allowed us to visualize interactions between different entities such as Firefighter, Intervention, Vehicle, and Fire Station.

Tkinter Application

The interface developed with Tkinter allowed to:

  • Query the firefighter database.
  • Add or modify information in the database.

Conclusion

This project allowed us to develop skills in data modeling, interface-oriented programming, and interaction between an application and a database. Working in a duo also taught us to collaborate effectively on a technical project.
