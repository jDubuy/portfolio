Project: SAE Data Collection

This project was completed in a group of four students as part of our training. Its objective was to introduce us to processing data from a website or API.

Project Objectives

  • Create a map from an API.
  • Generate charts from data obtained from this API.
  • Extract information from a web page.
  • Produce charts from extracted data.

Context

For this project, we used Python and divided our group into two teams. The first was responsible for API management and had to produce a map of residential energy consumption for the cities of La Rochelle, AngoulÃªme, Poitiers, and Niort, as well as an associated chart. The second team had to choose a web page containing geographic information, extract the data, and create a map. We chose the Wikipedia page on European volcanoes, from which we created several charts.

Conclusion

Overall, the project went well and we encountered no major difficulties. It allowed us to better understand how APIs work, as well as discover Python modules for extracting data from web pages.
